# AI동행 업로드 가이드

## 📁 파일 구조
```
/
├── index.html              ← 메인 사이트 파일 (교체)
└── images/
    ├── logo.svg            ← 로고 원본 SVG
    ├── favicon_16.png      ← 파비콘 16px
    ├── favicon_32.png      ← 파비콘 32px (브라우저 탭)
    ├── favicon_48.png      ← 파비콘 48px
    ├── favicon.png         ← 파비콘 192px (메인)
    ├── apple_touch_icon.png← iPhone 홈화면 아이콘 (180px)
    ├── icon_192.png        ← Android/PWA 192px
    ├── icon_512.png        ← Android/PWA 512px
    ├── og_image.png        ← SNS/카카오 공유 썸네일 (1200×630)
    ├── kakao_profile.png   ← 카카오채널 프로필 (640×640)
    ├── kakao_cover.png     ← 카카오채널 커버 (1920×640)
    ├── andong_watercolor.png
    ├── andong_village.png
    └── andong_ink.png
```

## ✅ GitHub 업로드 순서
1. `index.html` → 루트에 업로드 (기존 파일 덮어쓰기)
2. `images/` 폴더 전체 → 루트의 images/ 폴더에 업로드

## 📱 카카오채널 적용
- 프로필: `kakao_profile.png` (640×640)
- 커버: `kakao_cover.png` (1920×640)
